# 내러티브 아크 가이드

제안서의 컨셉(Phase 3)은 단순한 키워드 나열이 아닌, 스토리텔링 아크를 따라야 합니다.

## 내러티브 아크 구조

### 1. 세계관 진입 (Hook)
- 브랜드/IP의 세계관에서 출발
- "만약 [IP]의 세계가 현실에 나타난다면?"
- 몰입감 있는 첫 문장

### 2. 갈등/기회 제시 (Tension)
- 현재 상황의 한계점
- "팬들은 [구체적 니즈]를 원하지만, 아직 충족되지 않았습니다"
- 데이터 기반 근거

### 3. 해결의 실마리 (Turning Point)
- 우리만의 차별화된 접근
- "우리는 [독창적 방법]으로 이 경험을 완성합니다"
- Win Theme과 연결

### 4. 비전 제시 (Resolution)
- 프로젝트 완료 후의 모습
- "이 프로젝트가 성공하면, [구체적 성과]를 달성합니다"
- 정량적 KPI와 연결

## 적용 방법

### Phase 3 (CONCEPT) 슬라이드 구성:
1. **Big Idea Reveal**: 컨셉 키워드 + 세계관 연결 (다크 배경, 대형 타이포)
2. **Strategy Map**: 3대 전략 축의 순환 구조 시각화
3. **Experience Journey**: 관람객/유저의 경험 여정을 시간순으로

### 주의사항:
- 컨셉은 반드시 IP/브랜드의 세계관에서 파생되어야 함
- 추상적 단어 대신 구체적 행동/경험으로 표현
- 3대 전략 축은 상호 연결되어 시너지를 보여야 함
